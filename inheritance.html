Inheritance

Two decades ago when Java was born, it brought a concept of Inheritance. What is inheritance and what are the rules of inheritance?

Inheritance allows a class to be a subclass of a superclass and thereby inherit public and protected variables and methods of the superclass. Inheritance is a key concept that underlies IS-A, polymorphism, overriding, overloading, and casting. All classes (except class Object) are subclasses of type Object, and therefore they inherit Object's methods.

Rules:

  1. Subclasses know their parent (super classes) that they inherit from but super classes doesn't know anything about the sub classes.
  2. An Interface can extend "n" number of interfaces / abstract classes
  3. An Abstract class can implement "n" number of interfaces but can only extend one Abstract class, in fact an Abstract class can as well extend a concerete class.
