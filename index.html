Pass-by-Value vs. Pass-by-Reference

Java Pass-by-Reference or Pass-by-Value

  1. Pass-by-Value: Pass-by-Value means: The called method can't change the value of the caller's method.

  2. Pass-by-Reference: Pass-by-Reference means: The called method can change the value of the caller's method.

When we talk about Pass-by-Reference vs. Pass-by-Value, we actually need to see what semantics the language is using.

Two things to understand.

  1. Java manipulate objects by their references.
  2. It passes references to methods by value.

Whether you are passing a primitive variable or the reference to an object; you are always passing a copy of bits in the variable. The called method then gets its own copy of the reference variable to do with it what it likes.But because two identical reference variables refer to the exact same object, if the called method modifies the object (by invoking setter methods, for example), the caller will see that the object the caller's original variable refers to has also been changed.

Note: Strings are an exception, in java strings are immutable and they always create a new object.

Example

public class JavaSemantics {
    public static void main(String[] args) {
            ClassRoom classRoom = new ClassRoom(5, 5, 5);
                  System.out.println("Before magic " + classRoom.getWidth() + " " + 
                         classRoom.getHeight() + " " + classRoom.getLength());

          // Passing the classRoom object as a reference to magic by value.
            magic(classRoom);
            System.out.println("After magic " + classRoom.getWidth() + " " + 
                   classRoom.getHeight() + " " + classRoom.getLength());
   }
}

static void magic(ClassRoom classRoom) {

    //Changing the original Object for which we got the reference as a value.
    classRoom.setWidth(10);
    classRoom.setHeight(10);
    classRoom.setLength(10);

    // Assigning classRoom reference to a new ClassRoom object.
    classRoom = new ClassRoom(20, 20, 20);

    System.out.println("Inside magic " + classRoom.getWidth() + " " + 
                               classRoom.getHeight() + " " + classRoom.getLength());
}
}

class ClassRoom {
    private int width;
    private int height;
    private int length;

public ClassRoom(int width, int height, int length) {
    this.width = width;
    this.height = height;
    this.length = length;
}

public void setWidth(int width) {
    this.width = width;
}

public void setHeight(int height) {
    this.height = height;
}

public void setLength(int length) {
    this.length = length;
}

public int getWidth() {
    return width;
}

public int getHeight() {
    return height;
}

public int getLength() {
    return length;
}
}



Output:
Before magic 5 5 5
Inside magic 20 20 20
After magic 10 10 10
